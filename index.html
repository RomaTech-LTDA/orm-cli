@romatech/orm-cli

GitHub npm

Command-line interface for @romatech/orm. Manage migrations, apply schema changes, and scaffold TypeScript entities directly from your terminal.

Installation

npm install -g @romatech/orm-cli

Quick Start

# Create a migration from current entity state
orm migration:create InitialCreate

# Apply all pending migrations
orm update-database

# Scaffold entities from existing database
orm scaffold

# Revert the last migration
orm downgrade-database

Features

Migration Management

Create, apply, and revert migrations. Supports targeting specific migration names.

Database Scaffold

Reverse-engineer TypeScript entity files and a DbContext from an existing database schema.

Any Provider

Works with all 7 ORM providers: SQL Server, MySQL, PostgreSQL, Oracle, MongoDB, SQLite, Memory.

Config File

Simple orm.config.js with provider, paths, and entity configuration.

JSON Migrations

Migrations stored as JSON files — human-readable, versionable, and diffable.

ESM Support

Uses dynamic ESM imports. Works with any TypeScript or JavaScript project setup.
